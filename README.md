# Quoridor
Ce projet est une implémentation complète du jeu de plateau **Quoridor**. Il inclut un moteur de jeu respectant les règles officielles, une interface graphique interactive et une IA capable de défier des joueurs humains.

## Installation
docker compose down && docker compose up --build

## Utilisation
- Frontend (jeu) : http://localhost:5173
- Backend (santé) : http://localhost:8000/health
- Documentation API : http://localhost:8000/docs
